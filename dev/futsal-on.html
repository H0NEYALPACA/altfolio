  • 프로젝트명
    풋살ON(풋살매칭사이트)
    프로젝트 기간
    2021년 12월 26일 ~ (팀프로젝트 - 3인)
    프로젝트 설명
    기존에 축구나 풋살 상대를 찾을 때 직접 발품을 팔아 전화나 문자로 매칭하는 과정에서 불편함을 느꼈고,
    보다 효율적이고 빠르게 상대를 매칭할 수 있는 시스템이 있었으면 좋겠다는 생각에서 시작된 프로젝트입니다.
    매칭 시스템을 가진 예약 사이트를 통해 더 활발한 스포츠 문화를 제공하고자 하였으며,
    매칭 시스템 뿐만 아니라 팀 매니징 기능을 통해 팀의 소속감과 전적을 확인할 수 있는 시스템을 개발하여 사용자에게 보다 흥미로운 서비스를 제공하고자 하였습니다.
    개발환경
    Skills Java / JPA / Thymeleaf / JavaScript / HTML / CSS
    Database Oracle
    Framework Spring Boot
    Library & API KAKAO Maps API / Javax mail

  • [1/5] 회원은 나만의 팀을 생성하거나 기존의 팀에 참가할 수 있습니다.
    팀 생성시 팀 이름 중복 여부를 체크하며 팀 생성이 완료되면 랜덤으로 팀 코드(UUID)를 부여하며 팀 가입시 팀 코드를 입력하여 가입하도록 하였습니다.
  • [2/5] 매치는 팀매치와 용병매치로 나뉘며, 현재 모집중인 매치만 리스트에 노출됩니다.
    신청하기 버튼을 클릭시 매치가 성사되며 성사된 매치는 팀 관리 또는 마이페이지에서 확인할 수 있습니다.
  • [3/5] 매치글 등록 페이지입니다.
    장소 키워드를 입력하면 Kakao API를 활용하여 장소를 검색하고 클릭시 해당 장소와 주소가 저장됩니다.
  • [4/5] 작성한 매치글은 나의 팀 - 글 관리 메뉴에서 확인할 수 있습니다.
    글 관리 메뉴에서 모집중인 게시글은 수정, 삭제가 가능하며 매치가 성사되어 모집완료된 글은 매칭 취소 버튼만 노출됩니다.
  • [5/5] 팀 전적 페이지에서는 나의 팀의 전적과 평점을 확인할 수 있습니다.
    또한 진행된 매치가 노출되어 경기 결과와 상대팀에 대한 평점을 입력할 수 있도록 구현하고 있습니다.
